RoomFinders API Documentation

API for finding rooms on campus.

Requests

  • Endpoints

    • /buildings - Returns all campus buildings.
    • /buildings?location=33.77689,-84.39779 - Returns campus buildings nearby a given location.
    • /buildings/{building_id} - Returns information about a single campus building.
    • /buildings/{building_id}/rooms - Returns the rooms for a given building.

  • Examples

    • https://roomfinders.herokuapp.com/buildings
    • https://roomfinders.herokuapp.com/buildings?location=33.77689,-84.39779
    • https://roomfinders.herokuapp.com/buildings/63
    • https://roomfinders.herokuapp.com/buildings/63/rooms

Responses

  • Format

    • All responses are returned in JSON format.

  • Status Codes

    • 200 OK - request was successfully returned.
    • 400 Bad Request - invalid request data.
    • 404 Not Found - requested data not found.
